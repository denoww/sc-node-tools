<h1>Instalação</h1>

Usando npm:

<pre>
  $ npm i sc-node-tools -s
</pre>

No npm:

<pre>
  require('sc-node-tools')

  scPrint.info('scNodeTools criado com sucesso')
</pre>

<h1>Atualizando pacote</h1>

Logando no npm:

<pre>
  npm login
</pre>

Usando npm script:

<pre>
  $ npm run sc:deploy -- tag=x.x.x msg='Commit Message'
</pre>
