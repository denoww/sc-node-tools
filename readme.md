<h1>Instalação</h1>

Usando npm:

<pre>
  $ npm i sc-util -s
</pre>

No npm:

<pre>
  require('sc-util')

  scPrint.info('scUtil criado com sucesso')
</pre>

<h1>Atualizando pacote</h1>

Logando no npm:

<pre>
  npm login
</pre>

Usando npm script:

<pre>
  $ npm run sc:deploy -- tag=x.x.x msg=Commit Message
</pre>
